# Beispieldaten

Geplante Datenquellen für den Running Case (Financial Crime / GraphRAG):

- **ICIJ Offshore Leaks** — öffentlich, Graph-Format (Offshore-Firmen, UBOs)
  <https://offshoreleaks.icij.org/>
- **OpenSanctions** — maschinenlesbare, täglich aktualisierte Sanktionslisten
  <https://www.opensanctions.org/>
- **Synthetische Daten** — fiktives Szenario "Silicon Shield" (siehe
  [`../docs/domain-demo-neo4j-sanctions.md`](../docs/domain-demo-neo4j-sanctions.md))

> Rohdaten werden **nicht** eingecheckt (siehe `.gitignore`); hier liegen nur
> Loader-Skripte und kleine, versionierbare Samples.
